Privacy Policy for Web Content Selector

Effective Date: 19/08/2025

Data Collection

Web Content Selector collects only the website content (text and HTML elements) that you explicitly select and choose to save. This includes:

  • HTML content of selected page elements
  • Text content of selected elements
  • Page URLs where content was captured
  • User-defined titles for saved content

Data Storage

All captured data is stored locally in your browser using Chrome's local storage API. No data is transmitted to external servers or third parties.

Data Control

You have complete control over your data:

  • Delete individual saved items
  • Delete all saved content
  • Export your data as JSON files

No Data Sharing

We do not collect, transmit, or share any personal information or user data with third parties.
